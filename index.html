About Me

Hi! I’m a senior at Cal studying Computer Science, Integrative Biology, and English. If you can’t tell from all my majors, I love learning and trying new things! I discovered computer science at the end of my junior year when on a whim I took CS61A, the introductory CS class at Cal. I loved the class so much that I decided that I would add the major no matter how late in the game it was and still graduate in four years! I hope to find ways to combine my multiple backgrounds and create software that can really help people.


Who am I?

We had four horses, two goats, five dogs, two cats, a pig, a handful of chickens, and then there was me, the youngest of nine children. My parents were a part of the Hare Krishna movement when they were younger so all of us ended up with Indian names. You'll have to ask me to list them off.

Living with so many animals probably first sparked my interest in biology and after a year at Cal I was set on the course to be a wildlife biologist, then a doctor, no, a vet. I simply could not settle on one course of life with biology. All I knew was that I wanted to do something that could impact people. When I started learning how to program I was so excited that I could finally build things and get some hands on experience!

I'm excited to move forward with my computer science career and continue to build new things. I am extremely interested in efficient algorithms, artificial intelligence, computational biology, and graphic design. There is a lot to learn, but I hope to find an awesome position as a Software Engineer and continue to build new hacks just for fun!

Personal Projects

Housemates:
House Manager Web Application

Housemates is a web application written in Python using the Flask microframework. I wanted to create a website that would help housemates manage the yuckier parts of living together, such as who does which chores and when. Housemates using my website can track bills and receive friendly text reminders when bill due dates are approaching. Each house receives its own private space to post house events and send messages to each other.

Class Projects

RHS
Simplify Regular Expressions

For my compilers class I designed and wrote a JavaScript DSL that helps users compose regular expressions using method chaining, function composition, and set operations, e.g. intersect, complement, union. The set operations functionality parses REs to create ASTs which are then converted into automata (NFAs and DFAs) that after applying the set operation are converted back into brand new REs that the user did not have to compose.

Firewall

For my networking class, I implemented a stateful active application-layer firewall in Python. My firewall generated packets in response to denied packets as well as log selected http transcations over TCP connections. The firewall first takes in a set of rules to apply to incoming and outgoing packet. Each IPv4 packet is parsed to determine the necessary fields such as source and destination addresses as well as protocol type (TCP, UDP, or ICMP) and the correct rule is applied to determine whether to drop or pass the packet. CS164 Interpreter

For my compilers class, I wrote a complete interpreter in Javascript for a simple language called CS164. It is capable of defining variables, functions, dictionaries, lists, lambdas, coroutines, objects with protocol based inheritance, and much more! In addition, it is also tail call optimized! File Compressor

For my data structures class, I wrote a compressor engine in Java that could compress any type of file or even directories. The engine first scans through the file byte-by-byte and determines the best encoding map with Huffman Encoding. Then the files stream into the compressor which encodes them. (email me for the code)

Sliding Block Puzzle Solver

For my final project in data structures. I built a puzzle solver in Java that takes in the initial game state of a puzzle and some type of goal. Then the program chooses the best option of graph searches to find the optimal solution. The graph searches I used were A*, Iterative deepening, depth-first search and combinations of these algorithms. I also wrote the heuristic functions to greatly prune the search states.

Pear-Soap:
House Manager Web Application

Pear-Soap is a web application written in Python using the Flask microframework. I wanted to create a website that would help housemates manage the yuckier parts of living together, such as who does which chores and when. Housemates using my website can track bills and receive friendly text reminders when bill due dates are approaching. Each house receives its own private space to post house events and send messages to each other.
